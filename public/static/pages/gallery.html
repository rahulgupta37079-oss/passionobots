Add Your Journey Images

To populate this gallery with your Google Drive images:
1. Make images publicly accessible or download them
2. Upload to a CDN or use direct image URLs
3. Update the gallery data in the JavaScript section below

View Instructions

How to Add Your Images

1

Download Images from Google Drive

Download all images from your Google Drive folder to your computer.

2

Upload to Image Hosting

Upload images to one of these services:

  • Cloudflare R2 (Recommended for this project)
  • Imgur - Free, simple image hosting
  • ImgBB - Free with direct links
  • GitHub - Store in repository assets folder
3

Update Gallery Data

Edit the galleryData array in the JavaScript section below:

const galleryData = [
  {
    id: 1,
    image: 'YOUR_IMAGE_URL_HERE',
    title: 'Robotics Workshop 2024',
    description: 'Students building autonomous robots',
    category: 'workshops',
    date: '2024-01-15'
  },
  // Add more images...
];
